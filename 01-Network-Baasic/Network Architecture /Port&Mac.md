網路基礎定址與分配 
1. MAC 位址 vs. IP 位址：
   分層職責
   | 項目 | MAC 位址 (Layer 2) | IP 位址 (Layer 3) |
   | :--- | :--- | :--- |
   | 類型 | 實體位址 (硬體 ID) | 邏輯位址 (網路地址) |
   | 變動性 | 固定不變 (設備身份證) | 動態可變 (連線到不同網路會變) |
   | 範圍 | 僅用於本地網段內傳輸 | 用於跨網段路由與全球連線 | - ARP 協定： 在本地網路中，將 IP 位址轉換為對應的 MAC 位址。

2. IP 位址的分配機制
   | IP 類型 | 誰來分配？ | 功能 |
   | :--- | :--- | :--- |
   | 私有 IP (Private IP) | DHCP 伺服器 (通常是路由器內建) | 分配給區域網路 (LAN) 內的終端設備。 |
   | 公眾 IP (Public IP) | 網路服務供應商 (ISP) | 整個網路對外連線的唯一地址。 |
